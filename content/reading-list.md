A curated collection of resources I find valuable for AI-assisted development, Claude Code, and building with LLMs.

## Claude & Anthropic

- [Claude Code Documentation](https://docs.anthropic.com/en/docs/claude-code) — Official docs for the Claude Code CLI
- [Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook) — Practical examples and guides for building with Claude
- [Claude Model Card](https://docs.anthropic.com/en/docs/about-claude/models) — Capabilities and context windows for each model

## AI-Assisted Development

- [Prompt Engineering Guide](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview) — Anthropic's guide to getting better results from Claude
- [Building Effective Agents](https://www.anthropic.com/engineering/building-effective-agents) — Anthropic's research on agentic AI patterns
- [AI-Native Development](https://blog.pragmaticengineer.com/ai-tooling-for-software-engineers/) — The Pragmatic Engineer on AI tools for developers

## Tools & Infrastructure

- [Hugo](https://gohugo.io/) — The static site generator powering this site
- [PaperMod](https://github.com/adityatelange/hugo-PaperMod) — The Hugo theme we use (heavily customized)
- [Vercel](https://vercel.com/) — Deployment platform with edge functions
- [Supabase](https://supabase.com/) — Open source Firebase alternative (PostgreSQL + Auth + Realtime)
- [Linear](https://linear.app/) — Project management built for software teams

## AI Coding Perspectives

- [The End of Programming as We Know It](https://www.oreilly.com/radar/the-end-of-programming-as-we-know-it/) — Tim O'Reilly on AI's impact on software development
- [llms.txt Standard](https://llmstxt.org/) — Making websites AI-readable

## Open Source

- [Claude Code on GitHub](https://github.com/anthropics/claude-code) — The CLI tool itself
- [Adventures in Claude Source](https://github.com/bradfeld/adventuresinclaude) — This site's source code
